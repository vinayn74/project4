# Project 4: Product Showcase

## Learning Objectives

### HTML Concepts:
1. **Tables**: Create a comparison table using `<table>`, `<tr>`, `<td>`, `<th>`
2. **Semantic Sections**: Use `<article>` for product cards and `<aside>` for sidebar
3. **Meta Tags**: Add proper meta tags for SEO and description

### CSS Concepts:
1. **CSS Grid**: Use grid layout for product cards
2. **Flexbox Advanced**: Use justify-content and align-items for alignment
3. **Animations**: Create keyframe animations
4. **Transform**: Use transform for hover effects (scale, rotate, translate)

## Instructions

1. Create showcase structure in `index.html`:
   - Add meta description tag
   - Create product cards using article elements
   - Add a comparison table
   - Use aside for additional information

2. Style the showcase in `styles.css`:
   - Use CSS Grid to layout product cards
   - Add hover animations with transform
   - Create keyframe animations for loading or effects
   - Style the comparison table
   - Use flexbox for card content alignment

## Expected Result
A product showcase page with:
- Grid layout of product cards
- Animated hover effects
- Comparison table
- Professional design
